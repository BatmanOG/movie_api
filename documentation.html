Movie Api

the point of this project is to familularize myself with creating an applications backend server and api through the guidance of careerfoundry. when completed this the user will be able to search for a movie by it genre, actors, or director and find other information on said movies. this will be done using powershell and node.js and multiple modules included with it and third party modules as well. users will be able to create a profile and and save information on movies they like or want to see.

I added Express framework to aid in the creation of the server and morgan to log it all in the log.txt file

in this app the user will be able to create a profile with a username and password to save their favorite movies and informtion about that movie and update their persona information. the user will also be able to look up movies by name, genre, year, actors, and director.

business logic URL HTTP Method Query Parameters Path Variables Response data
Return list of all movies /movies GET/Read none none json array holding all info on the movies.
Return information on specific movie /movies/:title GET/Read [title] {title} { Title:'movieTitle', Year:'xxxx', certificate:'x', Time:'runTime', genre:['genre'], textmuted:'description.', Director:'directorName', Stars:['StarNames'] }
Return a list of movies by a specific director /movies/director/:directorName GET/Read [directorName] :directorName { Title:'movieTitle', Year:'xxxx', certificate:'x', Time:'runTime', genre:['genre'], textmuted:'description.', Director:'directorName', Stars:['StarNames'] }
Return information on a specific director /Directors/:director GET/Read [Director] :director { directorName: 'Name', Birthdate: 'Month Day, Year', Birthplace: 'location', biography:`Discription` }
Return list of movies and their info from a certian genre /movies/genre/:genreName GET/Read [genre] {genreName} { Title:'movieTitle', Year:'xxxx', certificate:'x', Time:'runTime', genre:['genre'], textmuted:'description.', Director:'directorName', Stars:['StarNames'] }
Creat new user /users POST/Create [newUser] none { id: id number, name: 'Name', favoriteMovies: [] }
update username /users/:id PUT/Update [id] {updatedUser} { id: id number, name: 'updatedName', favoriteMovies: [] }
Create favorite movie list /users/:id/:title POST/Create [ id, title] none (`${title} has been added to user ${user.name}'s favorites`)
Remove movie from favorites list /users/:id/:title DELETE/Delete [ id, title] none (`${title} has been remove from user ${user.name}'s favorites`)
Remove user /users/:id DELETE/Delete [ id ] none (`user ${user.name} has been deleted`)
